# Page Map

## Required pages
1. `/` — Homepage
2. `/starter-kit` — Starter Kit opt-in page
3. `/thank-you` — Thank-you page
4. `/guide` — Guide sales page
5. `/lab` — AI Employee Lab membership page

## Homepage purpose
Orient the brand and route users to the right offer.
Primary CTA: Starter Kit.

## Starter Kit page purpose
Capture emails.
One job only.

## Thank-you page purpose
Confirm opt-in and bridge to guide.

## Guide page purpose
Convert to first purchase.

## Lab page purpose
Convert to recurring membership.

## Shared components needed
- Navbar
- Footer
- Hero section
- CTA block
- Feature grid
- Pricing cards
- FAQ accordion
- Email form block

## Recommended build order
1. Starter Kit page
2. Thank-you page
3. Guide page
4. Lab page
5. Homepage
